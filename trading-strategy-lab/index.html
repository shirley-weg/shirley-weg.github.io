目前已整理好的功能

第一版先把 notebook 裡的 pair trading 策略整理成可重用的網站 app，後續可以把 TXO VIX-like、期貨策略、投組最佳化與 rebalancing 加進同一個策略庫。

Pair Trading Backtest

使用 rolling OLS 估計 spread 與 z-score，沿用 t 日 Close 產生訊號、t+1 日 Open 進出場。

Sharpe 權重最佳化

除了 OLS hedge ratio，也可用 rolling grid search 最大化歷史 Sharpe，決定兩檔資產 gross weight。

互動圖表與交易明細

呈現權益曲線、benchmark 比較、drawdown、spread signal、entry weights 與 trade P&L。

架構方向

GitHub Pages 負責這個靜態入口頁；真正的 Python 回測服務需要 Streamlit Cloud、Render 或 Railway 這類可執行 Python 的服務。

資料來源股票使用 yfinance；TX/TXO 預留 TAIFEX adapter。
策略層每個策略拆成可重用函式，未來可加入 portfolio optimization 與 rebalance frequency。
展示層Streamlit app 提供參數輸入、圖表與 CSV 下載。

部署方式

要讓使用者真的在網頁上按鈕跑回測，請把下列 main file 指到 Streamlit Cloud。GitHub Pages 本身不能執行 Python。

Branch: main
